### 页面生成
- [ ] 每页标题是否存在

### AI 编辑（7 个指令）
| 指令 | 预期 operation | 验证项 |
|------|---------------|--------|
| 把标题改短一点 | update_text | 标题变短，其他不变 |
| 这一页文字减少 50% | batch_update_text | 多个 block 文字缩短 |
| 把右侧卡片往左挪一点 | move_element | position.x 减小 |
| 把图片放大一点 | resize_element | width/height 增大 |
| 删除这个模块 | delete_element | block 从 content 消失 |
| 新增一个底部结论条 | add_element | content 增加 1 个 block |
| 改成左右对比布局 | replace_layout | layout 字段变化 |
